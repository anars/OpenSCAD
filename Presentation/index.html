OpenSCAD Workshop

The Programmers Solid 3D CAD Modeller

by Kay Anar ( @CoderKid )

What is OpenSCAD

  • 3D (three-dimensional) modeling tool
  • Open source
  • Cross Platform
  • First released in 2009
  • Uses programmatic Computer-aided design (CAD) style

What is Programmatic CAD?

  • Design paradigm
  • Objects are described using code
  • Easy to customize using parameters
  • Easy to share
  • Leads to good revision control

Programmatic CAD Softwares...

OpenSCAD

The most well known programmatic CAD program
Courtesy of Jason Lewis ( http://emacstragic.net/ )

OpenJSCAD

OpenSCAD + JavaScript = OpenJSCAD, edit locally or remote online, preview in browser direct via WebGL.
Courtesy of Jason Lewis ( http://emacstragic.net/ )

CoffeeSCad

Solid CAD modeling in your browser, with a CoffeeScript based syntax (and lots more!)
Courtesy of Jason Lewis ( http://emacstragic.net/ )

ImplicitCAD

Powerful, Open-Source, Programmatic CAD. Haskell and extopenscad.
Courtesy of Jason Lewis ( http://emacstragic.net/ )

RapCAD

Rapid prototyping CAD IDE for RepRap and RepStrap 3D printing machines.
Courtesy of Jason Lewis ( http://emacstragic.net/ )
